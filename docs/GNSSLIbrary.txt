GNSSLibrary
Moduł GNSSLibrary zapewnia funkcje do obsługi odbiornika GNSS (Global Navigation Satellite System). Pozwala na łączenie się z odbiornikiem, odczytywanie danych satelitarnych i przetwarzanie sygnałów GNSS.

Instalacja
Aby zainstalować moduł GNSSLibrary, wykonaj następujące polecenie:

shell
Copy code
pip install GNSSLibrary
Użycie
Importowanie modułu
Aby rozpocząć korzystanie z modułu GNSSLibrary, należy go najpierw zaimportować w swoim kodzie:

python
Copy code
from GNSSLibrary import GNSSLibrary
Inicjalizacja obiektu GNSSLibrary
Po zaimportowaniu modułu GNSSLibrary można utworzyć obiekt GNSSLibrary, który będzie służył do komunikacji z odbiornikiem GNSS:

python
Copy code
gnss = GNSSLibrary(port, baudrate)
port: Nazwa portu szeregowego, do którego jest podłączony odbiornik GNSS.
baudrate: Prędkość transmisji danych (np. 9600, 115200).
Odczyt danych satelitarnych
Aby odczytać dane satelitarne z odbiornika GNSS, użyj metody read_data():

python
Copy code
data = gnss.read_data()
Metoda read_data() zwraca odczytane dane w postaci słownika, zawierającego informacje o położeniu, prędkości, czasie i innych parametrach GNSS.

Przetwarzanie sygnałów GNSS
Moduł GNSSLibrary oferuje również funkcje do przetwarzania sygnałów GNSS, takie jak obliczanie pozycji, prędkości czy czasu. Oto przykład:

python
Copy code
position = gnss.calculate_position()
velocity = gnss.calculate_velocity()
timestamp = gnss.get_timestamp()
Metody te zwracają obliczone wartości na podstawie danych satelitarnych odczytanych z odbiornika GNSS.

Przykład użycia
Oto przykład, jak można użyć modułu GNSSLibrary w swoim projekcie:

python
Copy code
from GNSSLibrary import GNSSLibrary

# Inicjalizacja obiektu GNSSLibrary
gnss = GNSSLibrary('/dev/ttyUSB0', 115200)

# Odczyt danych satelitarnych
data = gnss.read_data()
print(data)

# Obliczanie pozycji
position = gnss.calculate_position()
print(position)

# Obliczanie prędkości
velocity = gnss.calculate_velocity()
print(velocity)

# Pobieranie czasu
timestamp = gnss.get_timestamp()
print(timestamp)
Ten przykład ilustruje podstawowe funkcje modułu GNSSLibrary, takie jak inicjalizacja obiektu, odczyt danych, przetwarzanie sygnałów GNSS i pobieranie czasu.
